Atalhos:

Para adicionar comentarios em uma linha use: (ctrl + Alt + ;)comentario no bloco, (ctrl + ;) cada linha. Os mesmo comando servem para descomentar.

Para adicionar uma nova linha em branco abaixo use: (ctrl + Enter).

Para adicionar uma nova linha em branco acima use: (ctrl + Shift + Enter).

Para alterar mais uma linha ao mesmo tempo use: (ctrl) e clicar nas linha desejadas.

Para alterar uma palavra que se repete varias vezes no codigo use: (ctrl + D).

Para unir o final de uma linha com o inicio da debaixa para eliminar espaço use: (ctrl + J).

Para exibir ou esconder a barra lateral de navegação de pasta use: (ctrl + KB).

Para pesquisar por titulo um documento que deseja ser editado use: (ctrl + P).

Para para dividir a tela e trabalhar com mais de um documento use: (Shift + Alt + 1 ou 2 ou 3 ou 4 ou 5 ou 8).

Para para alterar o foco entre os documentos use: (Ctrl + 1 ou 2 ou 3 ou 4 ou 5 ou 8).

Para alterar o documento atual de grupo de trabalho use: (Ctrl + Shift 1 ou 2 ou 3 ou 4 ou 5 ou 8).

Para altarar a identação por tab use: (Ctrl + Shift + P) e digite tabs.

Para saber onde inicia e termina as chaves do html use: (Ctrl + M).

Para para ordenar um texto em ordem numerica ou alfabetica use: (F9).

Para para manual ordenar um texto em ordem numerica ou alfabetica use: (Ctrl + Shift + seta).
